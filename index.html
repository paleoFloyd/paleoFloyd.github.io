Vim

  • 视频教程
    https://www.bilibili.com/video/BV1NG4y1p74h

插入

  • a (append 光标后插入)

  • i (insert 光标前插入)

  • o (open a line below 下一行插入)

  • A (append after lin 行最后插入)

  • I (inster before line 行最前插入)

  • O (append line above 在前一行插入)

  • gi 快速跳转到你最后一次编辑的地方并进入插入模式

插入模式下快速纠错

  • ctrl+h 删除上一个字符
  • ctrl+w 删除上一个单词
  • ctrl+u 删除当前行
  • 在终端也适用

visual模式 （一般用来选择文本）

  • normanl 模式下按v进入
  • V 选择行
  • ctrl+v 进行方块选择

移动

  • w/W 移动到下一个word/WORD开头
  • e/E 移动到下一个word/WORD结尾
  • b/B 回到上一个单词开头,backword
  • word 指的是以非空白符分割的单词，WORD以空白符分割的单词

行间搜索移动

  • 使用f{char} 可以移动到char字符上, t移动到char的前一个字符
  • 如果第一次没搜到，可以用;/, 继续搜改行的下一个/上一个
  • 大写的F表示反过来搜前面的字符

vim水平移动

  • 0移动到行首的第一个字符，^移动到第一个非空白字符
  • $移动到行尾, g_移动到行尾非空白字符

vim垂直移动 在句子和段落间移动(不常用)

  • 使用() 在句子间移动，你可以用:help ( 来查看帮助
  • 使用{} 在段落之间移动
  • 使用easy-motion 插件移动

vim页面移动

  • g/G移动到文件开头和结尾, 你可以用ctrl+o快速返回
  • H/M/L 跳转到文件的开头、中间和结尾
  • ctrl+u ctrl+f 上下翻页。zz把屏幕设置为中间

vim快速增删改查

vim快速删除

  • x普通模式下快速删除
  • d配合文本对象快速删除一个单词daw (delete around world)
  • d 和 x 都可以搭配数字执行多次
  • dt) 快速删除直到 ) 可替换为其他字符

vim快速修改

  • r(replace) c(change) s(subsitude)
  • noarmal 模式下可使用r 替换一个字符， s 替换并进入插入模式
  • c 配合文本对象可快速进行修改
  • R 不断替换后面一个字符
  • S 删除整行并进入插入模式
  • C 删除整行并进入插入模式（光标位置不变）
  • 数字+r/s 对多个字符进行修改

vim查询

  • 使用/或者?进行向前或者反向搜索
  • 使用n/N跳转到下一个或上一个匹配
  • 使用*或#进行当前单词的前向或后向匹配

vim替换命令

subsitude 允许我们替换文本并且支持正则

  • :[range]s/[ubstitute]/{partern}/{string}/[flags]
  • range表示范围如 :10,20 表示10到12行，%表示全部
  • partern 是替换的模式, string是替换后文本

flag 常用标志

  • g(global)全局范围内执行
  • c(confirm)可以确认或拒绝修改

vim多文件操作

buffer 切换

  • 使用:ls 会列举当前缓冲区， 然后使用:b n 跳转到第n个缓冲区
  • :bpre :bnext :bfirst :blast
  • :b buffer_name 加上tab补全来跳转

windown 窗口

  • <ctl+w>s 水平分割 <ctl+w>v 垂直分割 或:sp :vs
  • w 在窗口间循环切换
  • h 切换到左边的窗口
  • j 切换到下边的窗口
  • k 切换到上边的窗口
  • l 切换到右边的窗口
